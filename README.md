# La Patisserie Management System
## CS-412 Visual Programming – Semester Project

## Login Credentials
| Role     | Username | Password     |
|----------|----------|--------------|
| Admin    | admin    | admin123     |
| Customer | customer | customer123  |

## Features
### Admin Access
- View Menu (105 Pakistani items with PKR prices)
- Place Order (Cart + Auto Bill)
- Daily Sales Report

### Customer Access
- View Menu
- Place Order (Cart + Auto Bill)

## Technology
- Language: C# (.NET 10)
- UI: Windows Forms
- Database: SQLite (Microsoft.Data.Sqlite)

## How to Run
1. Open PatisserieMS.csproj in Visual Studio 2022
2. Press F5
3. Login with admin/admin123

## Database
SQLite database (patisserie.db) is created automatically on first run.
105 menu items are seeded automatically.
